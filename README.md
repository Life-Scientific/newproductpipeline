# Product Pipeline - Life Scientific Portfolio Management

This is a [Next.js](https://nextjs.org) project for managing agrochemical formulation portfolios through their entire lifecycle.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
